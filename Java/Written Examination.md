1、若要确定两个条件是否有一个为true或都为true时，可用逻辑异或^(错误)   
* 逻辑非(!)：0变1,1变0；简单记忆就是取反；  
* 逻辑与(&)：只要有一边为fale, 那么就是false    
* 逻辑或(|)：只要有一边为true, 那么就是true  
* 逻辑异或(^)：意思是排斥相同，即只有一个为0，一个为1时，结果为1，其他为0.  
* 双与(&&):一旦判断到左边是false之后, 右边就不参与运算了  
* 双非(||):一旦判断左边是true之后, 右边就不参与运算了
